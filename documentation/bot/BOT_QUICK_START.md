## 1. Starting the Bot (30 seconds)

### Find the Bot

**On Mobile:**
1. Open Telegram app
2. Tap the search icon (ğŸ”)
3. Type: `@YourSurveyBot`
4. Tap on the bot when it appears

**On Desktop:**
1. Open Telegram Desktop
2. Click search box at top
3. Type: `@YourSurveyBot`
4. Click on the bot

### Start Conversation

1. You'll see a **"START"** button at the bottom
2. Click it (or type `/start`)
3. Bot sends welcome message
4. You're registered automatically!

**Welcome Message:**
```
Hello, John!

Welcome to SurveyBot! I can help you create and manage surveys,
or participate in surveys created by others.

What would you like to do?
- Create and manage your own surveys
- Take surveys shared by others
- View results and statistics

Use the buttons below to get started, or type /help to see all available commands.

[Find Surveys]
[My Surveys] [Help]
```

**That's it!** You're ready to use SurveyBot.

## 5. Pro Tips

### For Survey Takers

**Read first:**
- Check estimated time
- Read all instructions
- Note which questions required (*)

**During survey:**
- Complete in one sitting
- Don't wait too long (30 min timeout)
- Review answers before submitting

**After completion:**
- View results if available
- Browse for more surveys
- Share feedback with creator

### For Survey Creators

**Planning:**
- Keep surveys short (5-10 questions)
- Mix question types
- Test before sharing

**Creating:**
- Use web interface (easier)
- Write clear questions
- Mark only essential questions as required

**Sharing:**
- Activate survey first
- Share link with respondents
- Monitor responses via bot

**Managing:**
- Check response count in `/mysurveys`
- Pause when done collecting
- View detailed stats on web

## 8. Troubleshooting Quick Fixes

### Bot Not Responding
**Fix:** Type `/start` and wait 10 seconds

### "Not Registered" Error
**Fix:** Type `/start` to register

### Can't Skip Question
**Fix:** Question is required (*), must answer

### Answer Too Long
**Fix:** Shorten to under 1000 characters

### Session Expired
**Fix:** Restart survey, complete in one sitting

### Survey Not Found
**Fix:** Survey may be deactivated, try `/surveys` for active surveys

**More help:** See [BOT_TROUBLESHOOTING.md](BOT_TROUBLESHOOTING.md)
